UKC dates for the Palm datebook

I live by their Palm organiser rather than the UKC paper diary. The diary has (at least) one advantage however: it specifies the week numbers. I have long found it a pain to enter all of these by hand into my organiser. So I've written a little Perl script to generate a set of regularly spaced dates (e.g. the dates of week starts) into a format that can then be converted into datebook .dba format. Here's the weeks for the terms 2006/07 in datebook .dba format.

To do this yourself, you'll need (1) Perl and (2) a little DOS tool called convdb. I got a copy from here. It's a pretty flexible if rather fragile tool for converting from plain text to the Palm .dba format.

Usage

events.pl [-h] [-l label] [-p period] [-s start] day/month/year repeat...
events.pl expects groups of pairs of arguments. It will generate a list of dates for each group, starting with the date given and repeated repeat times.

Options:

-h Print this message and exit.
-l label Print each event as "label N" where N is the number of the event.Default: "UKC Week".
-p period The repeat period. Default: 7
-s start The starting number for the label. Default: 1.

Example

First, generate a plain text listof dates in a format convdb can read. E.g.

events.pl 26/9/2005 12 9/1/2006 12 1/5/2006 6 > ukc0506.txt

will produce a list in plain text for the the three terms of 2005/6 academic year. You then need to convert it to .dba format with convdb; the .dba file will be called output.dba. (Yes, I know the Summer Term actually starts on 2/5/2006 but that's a Tuesday).

Rename output.dba to something sensible, e.g. ukc0506.dba and import it to your organiser (e.g. via the Palm desktop).
